## 1. プロジェクト概要決定事項

| 項目 | 決定内容 |
|------|----------|
| 目的 | 競馬の複勝馬券の期待値を算出し、長期的な回収率プラスを目指す |
| 予測ターゲット | 複勝（3着以内）：的中=1、不的中=0の2値分類 |
| 対象レース | JRA（中央競馬）平地レースのみ。障害レースは除外 |
| アルゴリズム | LightGBM（勾配ブースティング決定木） |
| 購入基準 | 期待値（予測複勝率 × 複勝オッズ）> 1.0 の馬のみ |
