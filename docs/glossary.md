## 2. 統計・機械学習用語

| 用語 | 説明 |
|------|------|
| 期待値 | 勝率×払い戻し倍率。1.0を超えれば長期的にプラス |
| 回収率 | 購入金額に対する払い戻し金額の割合（%） |
| AUC | モデルの識別能力の指標（0〜1）。0.65以上が目標 |
| LightGBM | 勾配ブースティング決定木の機械学習ライブラリ |
| 過学習 | 学習データに最適化しすぎて未知データに対応できなくなること |
| 特徴量 | モデルの入力となる変数 |
| バックテスト | 過去データでシステムの有効性を検証すること |
| 時系列分割 | 過去で学習・未来でテストする分割方法 |
| データリーク | 未来の情報が学習データに含まれてしまう問題 |
| upsert | データが存在すれば更新、なければ挿入するDB操作 |

## 3. システム用語

| 用語 | 説明 |
|------|------|
| Supabase | PostgreSQLベースのクラウドDBサービス |
