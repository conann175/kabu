# 競馬予測システム 設計書
## 複勝期待値分析モデル

---

## 1. 概要

JRA平地レースを対象に、複勝（3着以内）の期待値を算出し、長期的な回収率プラスを目指す予測モデルです。

### 分析対象

| 項目 | 内容 |
|------|------|
| 対象 | JRA（中央競馬）のみ |
| レース種別 | 平地レース（芝・ダート）のみ |
| 除外 | 障害レース |
| 予測ターゲット | 複勝（3着以内に入るか：1 or 0） |
| 最終目的 | 複勝オッズ × 予測勝率 による期待値算出 |

---

## 2. システム構成

| ツール | 用途 | 費用 |

## 3. 特徴量設計

### レース条件
| 特徴量 | 優先度 |
|--------|--------|
| 開催場所（10場） | 高 |
| 距離 | 高 |
| 芝/ダート | 高 |
| 馬場状態（良/稍重/重/不良） | 高 |
| クラス（G1/G2/G3/OP/条件戦） | 高 |
| 天気 | 中 |
| 枠番 | 中 |
| 出走頭数 | 中 |

### 馬の過去成績
| 特徴量 | 優先度 |
|--------|--------|
| 過去3走・5走の着順平均 | 高 |
| 過去3走・5走の複勝率 | 高 |
| 同距離での複勝率 | 高 |
| 同芝/ダートでの複勝率 | 高 |
| 前走着順 | 高 |
| 前走からの間隔（日数） | 中 |
| 前走との距離差 | 中 |
| 馬体重・前走比増減 | 中 |

### 騎手
| 特徴量 | 優先度 |
|--------|--------|
| 全体複勝率 | 高 |
| 距離別・場所別・馬場状態別複勝率 | 中 |

### 調教師
| 特徴量 | 優先度 |
|--------|--------|
| 全体複勝率 | 中 |
| 距離別複勝率 | 低 |

### 血統
| 特徴量 | 優先度 |
|--------|--------|
| 父の産駒複勝率（全体・距離別・芝ダ別） | 中 |
| 母父の産駒複勝率 | 低 |

### オッズ・人気
| 特徴量 | 優先度 |
|--------|--------|
| 単勝オッズ | 高 |
| 人気順 | 高 |
| 人気順÷出走頭数 | 高 |

---

## 4. モデル構築ステップ

| Step | 内容 |
|------|------|
| 1 | データ収集（2〜3年分） |
| 2 | SQLビューで特徴量集計 |
| 3 | LightGBMで複勝予測モデル学習 |
| 4 | バックテストで回収率検証 |
| 5 | 期待値算出・購入候補出力 |
| 6 | 本番運用（毎週手動実行） |

---

## 5. 期待値算出式

```
期待値 = 予測複勝率 × 複勝オッズ
期待値 > 1.0 → 買い
```

---

## 6. 懸念点

- データ量：最低2〜3年分（約14,000レース）必要
- 過学習：バックテスト精度に過度に依存しない
- オッズ変動：直前オッズと予測時のオッズが異なる場合あり
- 競馬は確率ゲーム：回収率100%超えを保証するものではない
